Welcome to Last Assassin

The game is simple; find and tag your target before you get tagged yourself. When you tag your target, they will join your team and you will track down the next target assigned to you. The last person left is the winner. Last Assassin can be played collectively on iPhone, Android, and on the web. Create a game session, invite your friends, and start playing Last Assassin.

Play on all devices

Last Assassin is entirely cross-platform. This means that a group of players can play a single game with iPhones, androids, and with any device that can open a browser (supported on Safari and Google Chrome). One player needs an iPhone or Android version of the game to start a game session (the game host), and any other device can join the game using a unique code given to the host upon game creation. Once everyone is in, the host can start the game and everyone is off to the races!
